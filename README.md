# CodeAlpha Frontend Development Tasks

## Overview
This repository contains the projects completed during my internship at CodeAlpha.  
The purpose of these tasks is to demonstrate practical knowledge of frontend web development using HTML, CSS, and JavaScript. Each project focuses on different aspects of user interface design, responsiveness, and interactive functionality.
---
## Tasks Completed

### 1. Image Gallery
A responsive image gallery that allows users to browse and view images in an organized layout.

**Features**
- Responsive grid-based layout
- Image hover effects
- Clean and simple UI design
- Smooth user interaction

**Technologies Used**
- HTML
- CSS
- JavaScript

---

### 2. Advanced Calculator
A web-based calculator designed to perform basic mathematical operations with an intuitive interface.

**Features**
- Addition, subtraction, multiplication, and division
- Square root calculation
- Real-time input display
- User-friendly interface

**Technologies Used**
- HTML
- CSS
- JavaScript

---

### 3. Music Player
A browser-based music player that allows users to control audio playback and navigate through a playlist.

**Features**
- Play and pause functionality
- Next and previous song controls
- Song title and artist display
- Progress bar for tracking playback
- Playlist sidebar
- Dark and light mode toggle

**Technologies Used**
- HTML
- CSS
- JavaScript
